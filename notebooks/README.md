# Notebooks

This directory contains Jupyter notebooks for experimentation and analysis.

## Notebooks

- **01_data_exploration.ipynb** - Explore the Food-101 dataset
- **02_model_training.ipynb** - Train the food classification model
- **03_evaluation.ipynb** - Evaluate model performance
- **04_nutritional_analysis.ipynb** - Analyze nutritional data

## Usage

Start Jupyter:

```bash
jupyter notebook
```

Or use JupyterLab:

```bash
jupyter lab
```
